# HyperOS Launcher Lite Mod (CN & EU Edition)

**Versions:**
- **RELEASE-5.39.35.10522-01062110** (CN Base)
- **RELEASE-5.39.35.10555-01211644** (EU Base)

HyperOS Launcher Lite Mod is a lightweight, performance-oriented launcher package designed specifically for **low-end Xiaomi, Redmi, and POCO devices**. This edition includes both the **CN** and **EU** variants of the official HyperOS Launcher, allowing users to choose the version that best suits their ROM.

Designed to deliver the premium HyperOS experience without the performance cost of resource-intensive visual effects, this Lite edition retains many flagship launcher features while removing heavy blur implementations for improved speed, smoother animations, and lower memory usage.

---

## Features

- 🚀 Official HyperOS Launcher (CN & EU Editions)
- ✨ Smooth unlock animation
- 📱 Fluid app opening and closing animations
- 🔍 Circle to Search support
- 📂 Stack-style and iOS-style Recent Apps layout
- ⚙️ Multiple launcher customization options
- ⚡ Optimized for low-end Xiaomi, Redmi, and POCO devices
- 🪶 Lightweight and performance-focused
- 🚫 Advanced blur features removed for better responsiveness
- 🔒 Compatible with **Magisk**, **KernelSU**, and **KernelSU Next**
- ♻️ Systemless installation

> **Note:** This is a Lite edition of the HyperOS Launcher. Advanced blur effects and certain premium visual features have been intentionally removed to deliver a faster and more stable experience on entry-level devices.

## Installation

### Magisk

1. Download the latest release.
2. Open the **Magisk** application.
3. Navigate to **Modules**.
4. Tap **Install from Storage**.
5. Select the downloaded ZIP file.
6. Reboot your device.

### KernelSU / KernelSU Next

1. Download the latest release.
2. Open the **KernelSU** or **KernelSU Next** Manager application.
3. Navigate to **Modules**.
4. Install the downloaded ZIP file.
5. Reboot your device.

## Known Issues

### 1. "Go back to previous app" option

After every reboot, the **Go back to previous app** feature may become enabled automatically even though its toggle appears disabled.

**Temporary Fix:**

- Enable the toggle once.
- Disable it again.

The setting will remain disabled until the next reboot.

### 2. Other Issues

If you encounter any additional bugs or compatibility issues, please open an Issue in this repository with your device model, Android version, and ROM version.
